# PiqueteDefend

Castle Wars-style 2-player local multiplayer game built with Unity.

## Overview

Two players on the same machine defend their castles and attack the opponent using various units and spells.

## Development

- **Engine:** Unity 6 (or 2022 LTS)
- **Players:** 2 local (split screen or shared screen)
- **Platform:** PC (Windows)

## Getting Started

1. Clone the repo
2. Open the project in Unity Hub
3. Open the `Main` scene from `Assets/Scenes/`
4. Press Play

## Project Structure

```
Assets/
  Scenes/       # Game scenes
  Scripts/      # C# game logic
  Prefabs/      # Reusable objects
  Art/          # Sprites and textures
  Audio/        # Sound effects and music
  UI/           # UI assets and canvases
```
